How the legs are chosen
Value first
Every leg starts with our model finding positive expected value against the bookmaker. If there's no edge, it doesn't make the acca.
One pick per match
No fixture appears twice. We pick the single best tip from each match — 1X2 and BTTS preferred over corners or handicaps.
Independent slates
Each acca uses a completely different set of matches. Faka Yonke takes the top 5 tips, Shaya Zonke takes the next 8, Istimela takes the rest — so a loss in one acca has zero impact on the others. The strongest tips are always in the smaller acca.
